# 🍐 FruitoplePack - Minecraft 1.12.2

**FruitoplePack** is a curated modpack that strikes a perfect balance between high-tech automation and a vibrant, living world. Build massive industrial empires beneath the canopy of trees with realistic physics!

## 🚀 Key Features

- ⚙️ **Advanced Tech:** Dive deep into industry with `GregTech`, `Ender IO`, `Mekanism`, and `Thermal Expansion`.
- 🌳 **Realistic Physics:** Experience the satisfying fall of timber (and cacti!) with `Dynamic Trees` integration.
- 🌌 **Space Exploration:** Journey through the solar system with `Galacticraft` and `Galaxy Space`.
- 🛠️ **Maximum Optimization:** Pre-configured with `Phosphor`, `BetterFPS`, and `FoamFix` for a smooth 200+ mod experience.

## 💻 System Requirements

- **RAM:** Minimum **5GB (5120MB)** allocation required.
- **Java:** Java 8 (JRE Legacy) 64-bit.
